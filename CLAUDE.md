# MA Construction - Baustellen-App

## Projektübersicht
PWA für Baustellenmanagement einer kleinen Innenausbau- und Renovierungsfirma in München.
Inhaber: Altin Malici. Zielgruppe: 2-10 Mitarbeiter + Subunternehmer.

## Tech Stack
- React mit Vite
- Tailwind CSS
- Lucide React Icons
- Single-File App (src/App.jsx)
- PWA, Mobile-first

## Features
- PIN-basierter Login (rollenbasiert: Chef, Mitarbeiter)
- Dashboard mit Modulen
- Baustellen-Verwaltung (anlegen, bearbeiten, löschen)
- Stundenerfassung mit Wetter
- Kalender und Terminplanung
- Regieberichte mit PDF-Vorschau
- Mängelmanagement
- Bautagebuch
- Kostenübersicht
- Materialübersicht
- Mitarbeiter- und Subunternehmer-Verwaltung

## Design
- Apple iOS Style, minimalistisch
- Lila Farbschema (Gradient #8E3A9E → #A04878)
- Mobile-first, responsive (Handy = native Look, Desktop = iPhone-Simulation)
- Deutsche UI-Texte

## Geplante Features
- Fotodokumentation (Vorher/Nachher)
- PDF-Export
- Materialtracking
- Digitale Unterschriften
- Offline-Fähigkeit

## Regeln
- Immer Mobile-first denken
- Große Touch-Buttons (min 44px)
- Deutsche Labels
- Arbeitszeiten auf 15-Minuten runden
- Bei Naturstein immer Steinart und Oberfläche angeben
